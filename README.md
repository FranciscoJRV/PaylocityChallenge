# PaylocityChallenge

Here you can find my Reports and The script to Test the APIs of the Paylocity Challenge

To be able to run the .jms file it is necessary to download Apache Jmeter.

You can run all Http Requests in one execution, or disable/enable the Http request you would like to.
In addition you can find the automation script, inside the main Java class (test/java/org.exmple/Main) you should type your won credentials and the employee information to be added/edited/deleted; Inside the Browser Java Class (test/java/Utilities/Browser) you should insert the path to your ChromeDriver executable.
